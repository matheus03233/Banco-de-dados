# Banco de Dados – ONG Solidária

Projeto de banco de dados relacional para a **Plataforma de Gestão da ONG Solidária**, baseado no minimundo e no modelo lógico desenvolvidos nas experiências práticas anteriores.

## Tecnologias

- Banco de Dados: MySQL (MySQL Workbench) – pode ser adaptado para PostgreSQL
- Linguagem: SQL (DML)
- Controle de versão: Git / GitHub

## Estrutura do repositório

- `00_criar_bd_tabelas.sql` – criação do banco `ong_solidaria` e das tabelas principais.  
- `01_inserts_ong.sql` – comandos `INSERT` para povoar as tabelas.  
- `02_selects_ong.sql` – consultas `SELECT` com `WHERE`, `ORDER BY`, `LIMIT`, `JOIN` e `GROUP BY`.  
- `03_updates_deletes_ong.sql` – comandos `UPDATE` e `DELETE` com condições.
